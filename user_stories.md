# User Story 1

**Title:**
_As an admin, I want to add new doctors to the system, so that they can manage patient appointments._

**Acceptance Criteria:**
1. Admin can enter doctor information.
2. Doctor data is saved successfully.
3. New doctor appears in the doctor list.

**Priority:** High
**Story Points:** 3

**Notes:**
- Doctor email must be unique.

---

# User Story 2

**Title:**
_As a patient, I want to book an appointment online, so that I can visit a doctor without calling the clinic._

**Acceptance Criteria:**
1. Patient can select a doctor.
2. Patient can choose an available time slot.
3. Appointment is stored successfully.

**Priority:** High
**Story Points:** 5

**Notes:**
- Only available slots should be displayed.

---

# User Story 3

**Title:**
_As a doctor, I want to view my appointment schedule, so that I can manage my daily workload._

**Acceptance Criteria:**
1. Doctor can see all upcoming appointments.
2. Appointment details are displayed correctly.
3. Schedule updates automatically after new bookings.

**Priority:** Medium
**Story Points:** 3

**Notes:**
- Past appointments should be separated from upcoming appointments.
